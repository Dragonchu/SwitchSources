# SwitchSources

## 介绍

SwitchSources是一个帮助开发者切换各种软件源的工具。

## 规划

- [ ] 支持maven, pip, npm等常见的开源工具的换源
- [ ] 支持用户自己配置源地址
- [ ] 提供pip安装包
- [ ] 提供brew安装包
- [ ] 支持Linux
- [ ] 支持Mac
- [ ] 支持windows
- [ ] 支持国产操作系统
- [ ] 支持测速功能
- [ ] 支持自动选择最快源
- [ ] 支持从远程拉取配置文件并使用
